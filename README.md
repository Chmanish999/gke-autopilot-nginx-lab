GKE Autopilot Cluster - Nginx Deployment

 Overview:
This project demonstrates how to create a Google Kubernetes Engine (GKE) Autopilot cluster and deploy a sample nginx application.

 Objectives:
- Create a GKE Autopilot cluster
- Deploy nginx workload
- Understand Kubernetes Deployment and Pods
- Explore logs, events, and YAML configuration

Technologies Used:
- Google Cloud Platform (GCP)
- Kubernetes (GKE Autopilot)
- Nginx

 Project Structure:
- README.md → Project explanation
- nginx-deployment.yaml → Kubernetes deployment file
- commands.txt → Useful commands
- notes.txt → Learning notes
- screenshots/ → Lab screenshots

Steps Performed:
1. Open Google Cloud Console
2. Navigate to Kubernetes Engine
3. Create Autopilot cluster
4. Go to Workloads
5. Deploy nginx application
6. Inspect Pods, Logs, and YAML

Learning Outcome:
This lab helped in understanding how Kubernetes manages applications using Deployments and Pods in a managed cloud environment.
